# NexaCart AWS Cloud Infrastructure

## Overview
A multi-region AWS cloud infrastructure designed and deployed for NexaCart, 
an e-commerce platform, as part of an Enterprise Cloud and Infrastructure 
Automation assignment.

## Services Used
- EC2, Application Load Balancer, Auto Scaling
- RDS Multi-AZ, S3, CloudFront, Route 53
- WAF, Shield, KMS, IAM, VPC

## Key Features
- Multi-region disaster recovery (ap-south-1 and ap-southeast-1)
- Applied AWS Well-Architected Framework across five pillars
- Infrastructure deployed using AWS CloudFormation
- 7Rs migration framework analysis

## Deployment
The infrastructure can be deployed using the CloudFormation template 
included in this repository via the AWS Console or AWS CLI.
